# 后端接口测试文档

> 版本：v1.0 | 日期：2026-03-15
> 项目：大学生心理健康咨询与"树洞"倾诉系统

## 测试环境

| 项目 | 版本/配置 |
|------|-----------|
| JDK | 21 |
| Spring Boot | 2.7.x |
| MySQL | 8.0+ |
| 数据库名 | `mental_health` |
| 服务端口 | `8080` |
| 测试工具 | cURL / PowerShell Invoke-RestMethod |

### 环境准备步骤

1. **初始化数据库**：执行 `database/schema.sql` 和 `database/init-data.sql`
2. **编译项目**：`cd backend && mvn clean compile -q`
3. **启动服务**：`mvn spring-boot:run`
4. **验证服务**：访问 `http://localhost:8080/api/article/list` 应返回 JSON

## 详细测试用例

### 模块一：认证模块

#### TC-01 用户注册
- **前置条件**：无
- **请求**：
```
POST /api/auth/register
Content-Type: application/json
{"username":"test2026@edu.cn","password":"test123"}
```
- **期望**：`code=200`，返回 `data.username="test2026@edu.cn"`, `data.role="STUDENT"`, `data.password=null`
- **反向测试**：重复注册同一用户名，期望 `code=500`

#### TC-02 用户登录
- **前置条件**：用户已注册
- **请求**：
```
POST /api/auth/login
Content-Type: application/json
{"username":"student001","password":"123456"}
```
- **期望**：`code=200`，`data.token` 非空，`data.userId` 非空

#### TC-03 登录失败（密码错误）
- **请求**：`{"username":"student001","password":"wrongpwd"}`
- **期望**：`code=500`，提示密码错误

#### TC-04 获取咨询师列表
- **请求**：`GET /api/auth/teachers`
- **期望**：`code=200`，返回至少 2 名咨询师信息，`password=null`

---

### 模块二：个人中心

#### TC-05 查看个人信息
- **前置条件**：已登录获取 Token
- **请求**：`GET /api/user/profile` + `Authorization: Bearer <token>`
- **期望**：`code=200`，返回用户信息（不含密码）

#### TC-06 修改密码
- **请求**：
```
POST /api/user/password
Authorization: Bearer <token>
{"oldPassword":"123456","newPassword":"654321"}
```
- **期望**：`code=200`，使用新密码可重新登录

#### TC-07 修改资料
- **请求**：
```
PUT /api/user/profile
Authorization: Bearer <token>
{"realName":"测试用户","gender":1}
```
- **期望**：`code=200`，再次 GET profile 验证字段已更新

### 模块三：科普文章

#### TC-08 文章列表分页
- **请求**：`GET /api/article/list?pageNum=1&pageSize=2`

#### TC-09 文章详情
- **请求**：`GET /api/article/1`
- **期望**：返回文章详细内容，连续两次访问 `views` 应递增
